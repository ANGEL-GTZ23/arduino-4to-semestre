📡 Arduino UNO con Sensor Ultrasónico HC-SR04 y Servo Motor

📡 Circuito: Arduino UNO con Sensor Ultrasónico HC-SR04 y Servo Motor

Figura 1 — Circuito con Arduino UNO, sensor de distancia HC-SR04 y servo motor. El servo gira según la distancia detectada por el sensor.

En esta imagen se muestra un circuito que combina tres componentes principales conectados a un Arduino UNO:

  • Sensor ultrasónico HC-SR04 (parte superior): mide la distancia a objetos emitiendo pulsos de sonido y detectando su eco. Tiene 4 pines: VCC (5V), GND (tierra), TRIG (disparo) y ECHO (eco), todos conectados al Arduino a través del protoboard.
  • Servo motor SG90 (parte inferior): motor de posición controlada. Recibe una señal PWM desde el Arduino que determina el ángulo al que gira (0° a 180°). Tiene tres cables: rojo (VCC 5V), negro/café (GND) y naranja/amarillo (señal de control).
  • Protoboard (breadboard): actúa como punto de distribución para los rieles de alimentación (rojo = 5V, negro/azul = GND), permitiendo conectar ambos dispositivos sin necesidad de soldar.
⚙️ ¿Cómo funciona este circuito?

El Arduino envía un pulso al pin TRIG del HC-SR04. El sensor emite una onda ultrasónica y espera el rebote en el pin ECHO. Con el tiempo que tarda el eco, el Arduino calcula la distancia. Según ese valor, envía una señal PWM al servo para girarlo proporcionalmente: objeto cerca → el servo gira a un ángulo menor; objeto lejos → el servo gira más. Es la base de sistemas de radar, puertas automáticas y robots de evitación de obstáculos.

🧰 Lista de materiales del circuito

Componente Cantidad Función
Arduino UNO 1 Microcontrolador principal; procesa datos y controla el servo
Sensor HC-SR04 1 Mide distancia por ultrasonido (2 cm – 400 cm)
Servo motor SG90 1 Gira de 0° a 180° según la distancia detectada
Protoboard (breadboard) 1 Distribuye la alimentación y organiza las conexiones
Cables Dupont (M-M / M-H) ~10 Conectan el Arduino, sensor y servo a la protoboard
Cable USB tipo B 1 Alimenta y programa el Arduino desde la computadora
🔌 Conexiones clave (pines del Arduino)

• HC-SR04 VCC → 5V del Arduino  |  HC-SR04 GND → GND
• HC-SR04 TRIG → Pin digital (ej. D9)  |  HC-SR04 ECHO → Pin digital (ej. D10)
• Servo señal → Pin PWM del Arduino (ej. D6)  |  Servo VCC → 5V  |  Servo GND → GND

💻 Lógica del código Arduino

Para programar este circuito en Arduino, la lógica es la siguiente:

  • Se incluye la librería #include <Servo.h> para controlar el servo motor fácilmente.
  • Los pines TRIG y ECHO del HC-SR04 se configuran como OUTPUT e INPUT respectivamente.
  • En el loop(), se envía un pulso de 10 microsegundos al pin TRIG con digitalWrite().
  • Se mide el tiempo que tarda el eco con pulseIn(echoPin, HIGH) y se convierte a centímetros dividiendo entre 58.
  • Con map(distancia, 2, 200, 0, 180) se convierte la distancia a un ángulo de 0° a 180°.
  • Finalmente, servo.write(angulo) mueve el servo a la posición calculada.
🤖 ¿Por qué es importante este circuito?

La combinación de un sensor ultrasónico con un servo motor es la base de proyectos como radares caseros, brazos robóticos reactivos, puertas automáticas y sistemas de evitación de obstáculos. Conecta conceptos de programación, electrónica, física del sonido y mecánica en un solo proyecto.
