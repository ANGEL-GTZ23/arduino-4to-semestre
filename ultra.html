📡 Arduino UNO con Sensor Ultrasónico HC-SR04 y LED Indicador

🖼️ Circuito: Arduino UNO + HC-SR04 + LED con resistencia

Figura 1 — Circuito con Arduino UNO, sensor de distancia HC-SR04 y LED verde con resistencia. El LED se enciende o parpadea según la distancia detectada por el sensor.

🔍 Explicación de la imagen

En la imagen se muestra un circuito completo montado en un protoboard (breadboard), conectado a un Arduino UNO. El circuito tiene tres elementos principales:

  • Sensor ultrasónico HC-SR04 (parte superior, azul): Es el componente más destacado. Tiene dos "ojos" circulares: uno emite una onda ultrasónica y el otro la recibe al rebotar en un objeto. Sus 4 pines (VCC, GND, TRIG, ECHO) se conectan a la protoboard mediante cables de colores — rojo (5V), negro (GND), amarillo (TRIG) y azul/blanco (ECHO).
  • LED verde (parte superior izquierda del protoboard): Pequeño componente luminoso que actúa como indicador visual. Está conectado en serie con una resistencia de 220Ω a 330Ω (la franja café/naranja visible en la protoboard) para limitar la corriente y proteger tanto el LED como el pin digital del Arduino.
  • Arduino UNO (parte inferior): El cerebro del sistema. Recibe la medición del sensor, la procesa en su microcontrolador ATmega328P y decide si encender o apagar el LED dependiendo de la distancia calculada. Se programa desde una computadora mediante el cable USB tipo B visible a la izquierda.
⚙️ ¿Cómo funciona este circuito?

El Arduino envía un pulso corto (10 µs) al pin TRIG del HC-SR04. El sensor lanza una ráfaga ultrasónica de 40 kHz invisible al oído humano. Cuando esa onda rebota en un objeto, el sensor devuelve una señal HIGH en el pin ECHO durante el tiempo exacto que tardó el sonido en ir y volver. El Arduino mide ese tiempo con pulseIn() y convierte el resultado a centímetros dividiendo entre 58. Luego, si la distancia es menor a un umbral definido (por ejemplo, 20 cm), enciende el LED verde como alerta. Si el objeto se aleja, el LED se apaga.

🔌 Conexiones clave (pines del Arduino)

• HC-SR04 VCC → 5V del Arduino  |  HC-SR04 GND → GND
• HC-SR04 TRIG → Pin digital D9  |  HC-SR04 ECHO → Pin digital D10
• LED ánodo (+) → Pin digital D7 (a través de resistencia)  |  LED cátodo (–) → GND

💻 Lógica del código Arduino

Para programar este circuito en Arduino, la lógica paso a paso es:

  • Se definen los pines: trigPin = 9, echoPin = 10, ledPin = 7.
  • En setup(), se configuran TRIG y LED como OUTPUT, y ECHO como INPUT.
  • En el loop(), se envía un pulso LOW–HIGH–LOW al pin trigPin de 10 microsegundos.
  • Se usa pulseIn(echoPin, HIGH) para medir el tiempo del eco en microsegundos.
  • La distancia se calcula con la fórmula: distancia = duracion / 58 (en centímetros).
  • Si distancia < 20, se ejecuta digitalWrite(ledPin, HIGH) para encender el LED.
  • Si la distancia es mayor, digitalWrite(ledPin, LOW) apaga el LED.
💡 Variantes posibles de este circuito

Con el mismo hardware puedes ampliar el proyecto fácilmente: usar múltiples LEDs de distintos colores para crear un medidor de distancia por niveles (verde = lejos, amarillo = cerca, rojo = muy cerca); añadir un buzzer piezoeléctrico que suene con mayor frecuencia según la proximidad (como un sensor de estacionamiento); o mostrar la distancia en cm en una pantalla LCD 16x2 en tiempo real.

🤖 ¿Por qué es importante este circuito?

El sensor HC-SR04 con un indicador LED es uno de los proyectos más fundamentales en electrónica con Arduino. Introduce conceptos de física del sonido, temporización de pulsos, lógica condicional y control de salidas digitales. Es la base de sistemas de alarmas de proximidad, asistencia de estacionamiento, robots que evitan obstáculos y medidores de nivel de líquidos.
